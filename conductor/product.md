# Product Definition: Goblin

## Overview
Goblin is an advanced, AI-driven Dark Web OSINT (Open Source Intelligence) platform designed for proactive breach monitoring, credential leak detection, and threat intelligence gathering. It empowers security researchers and organizations to identify exposed data across the clear web, dark web (.onion sites), and specialized leak repositories.

## Target Audience
- Security Researchers & Penetration Testers
- SOC (Security Operations Center) Analysts
- Organizations monitoring for brand/employee data leaks
- Privacy-conscious individuals tracking their digital footprint

## Core Value Propositions
- **Automated Dark Web Discovery:** Native Tor integration with advanced circuit rotation for resilient deep-web scraping.
- **Immediate Leak Monitoring:** Real-time tracking of paste sites, dump bins, and Telegram log aggregators with adaptive Cloudflare bypass.
- **AI-Powered Analysis:** Multi-model LLM support (OpenAI, Anthropic, Ollama) for intelligent data extraction and classification.
- **Comprehensive Search:** Unified engine querying multiple dark-web search endpoints.
- **Breach Correlation:** Integration with HIBP, IntelX, and other breach repositories.

## Key Goals
1. Provide a "one-stop" interface for deep-web and leak investigation.
2. Automate the detection of credential patterns (API keys, PII, passwords).
3. Offer actionable intelligence through AI-summarized breach reports.
4. Maintain a low footprint and high anonymity through Tor rotation.
